# slacketron

Electron wrapper for Slack, built with Nativefier. Focused on Linux arm64 support.

## Dependencies

- `npm`
- `nativefier` >= 10.1.0 (via npm: `sudo npm i -g nativefier`)

## Building

For now, just run `./build.sh`.
